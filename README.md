# openvpnPHP
php library for managing openvpn connections




this library needs "expect" and "openvpn3" for intracting with os

you can only use it in linux base systems as it contains linux shellscripts

add your openvpn config file to openvpnconfig.ovn and your username and password to config file

you need to provide one of for options below :
<ul>
  <li>c) to connect</li>
  <li>p) to pause connection</li>
  <li>r) to restart connection</li>
  <li>d) to disconnect </li>
</ul>
